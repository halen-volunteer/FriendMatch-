## 二、认证与安全

### 2.1 密码加密 - BCrypt

**依赖**：
```xml
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-crypto</artifactId>
    <version>6.0+</version>
</dependency>
```

**特性**：
- 工作因子：12（默认）
- 加密时间：~100ms
- 防护：彩虹表攻击、GPU破解

**使用**：
```java
BCryptPasswordEncoder encoder = new BCryptPasswordEncoder();
String encoded = encoder.encode("rawPassword");
boolean matches = encoder.matches("rawPassword", encoded);
```

### 2.2 敏感词检测

**依赖**：
```xml
<dependency>
    <groupId>com.github.houbb</groupId>
    <artifactId>sensitive-word</artifactId>
    <version>0.13+</version>
</dependency>
```

**用途**：
- 用户名敏感词检测
- 团队名称敏感词检测
- 消息内容敏感词检测

**使用**：
```java
SensitiveWordBs sensitiveWordBs = SensitiveWordBs.newInstance()
    .ignoreCase(true)
    .ignoreWidth(true)
    .init();

boolean contains = sensitiveWordBs.contains("用户输入");
```

## 三、邮件服务

### 3.1 Spring Mail

**依赖**：
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-mail</artifactId>
    <version>4.0+</version>
</dependency>
```

**配置**（application.yml）：
```yaml
spring:
  mail:
    host: smtp.qq.com
    port: 587
    username: your-email@qq.com
    password: your-app-password
    properties:
      mail:
        smtp:
          auth: true
          starttls:
            enable: true
            required: true
```

**用途**：
- 注册验证码邮件
- 找回密码邮件
- 系统通知邮件

**使用示例**：
```java
@Service
public class EmailService {
    
    @Autowired
    private JavaMailSender mailSender;
    
    public void sendVerificationCode(String email, String code) {
        SimpleMailMessage message = new SimpleMailMessage();
        message.setFrom("noreply@friendmatch.com");
        message.setTo(email);
        message.setSubject("FriendMatch 验证码");
        message.setText("您的验证码是：" + code);
        mailSender.send(message);
    }
}
```

## 六、实时通信 - WebSocket

### 6.1 Spring WebSocket

**依赖**：
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-websocket</artifactId>
    <version>4.0+</version>
</dependency>
```

**配置**：
```java
@Configuration
@EnableWebSocket
public class WebSocketConfig implements WebSocketConfigurer {
    
    @Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry registry) {
        registry.addHandler(chatWebSocketHandler(), "/ws/{userId}")
                .setAllowedOrigins("*");
    }
    
    @Bean
    public WebSocketHandler chatWebSocketHandler() {
        return new ChatWebSocketHandler();
    }
}
```

**处理器**：
```java
@Component
public class ChatWebSocketHandler extends TextWebSocketHandler {
    
    private static final Map<String, WebSocketSession> sessions = new ConcurrentHashMap<>();
    
    @Override
    public void afterConnectionEstablished(WebSocketSession session) {
        String userId = extractUserId(session);
        sessions.put(userId, session);
    }
    
    @Override
    protected void handleTextMessage(WebSocketSession session, TextMessage message) {
        // 处理消息
    }
    
    @Override
    public void afterConnectionClosed(WebSocketSession session, CloseStatus status) {
        String userId = extractUserId(session);
        sessions.remove(userId);
    }
}
```

## 七、测试框架

### 7.1 JUnit 5 + Mockito

**依赖**（Spring Boot 默认包含）：
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-test</artifactId>
    <version>4.0+</version>
    <scope>test</scope>
</dependency>
```

**单元测试示例**：
```java
@SpringBootTest
class UserServiceTest {
    
    @MockBean
    private UserMapper userMapper;
    
    @Autowired
    private UserService userService;
    
    @Test
    void testUserLogin() {
        // 准备测试数据
        User user = new User();
        user.setId(1L);
        user.setUserAccount("1234567890");
        
        // Mock 数据库查询
        when(userMapper.selectById(1L)).thenReturn(user);
        
        // 执行测试
        User result = userService.getUserById(1L);
        
        // 断言
        assertNotNull(result);
        assertEquals("1234567890", result.getUserAccount());
    }
}
```

## 八、构建与部署

### 8.1 Maven 构建

**pom.xml 基础配置**：
```xml
<project>
    <modelVersion>4.0.0</modelVersion>
    <groupId>com.zero</groupId>
    <artifactId>friendmatch</artifactId>
    <version>1.0.0</version>
    <packaging>jar</packaging>
    
    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>4.0.0</version>
    </parent>
    
    <properties>
        <java.version>21</java.version>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>

```

**常用命令**：
```bash
# 清理构建
mvn clean

# 编译
mvn compile

# 打包
mvn package

# 运行测试
mvn test

# 跳过测试打包
mvn package -DskipTests

# 运行应用
java -jar target/friendmatch-1.0.0.jar
```

**Dockerfile**：

### 9.2 异步处理 - Spring Async

**配置**：
```java
@Configuration
@EnableAsync
public class AsyncConfig {
    
    @Bean(name = "taskExecutor")
    public Executor taskExecutor() {
        ThreadPoolTaskExecutor executor = new ThreadPoolTaskExecutor();
        executor.setCorePoolSize(10);
        executor.setMaxPoolSize(20);
        executor.setQueueCapacity(100);
        executor.setThreadNamePrefix("async-");
        executor.initialize();
        return executor;
    }
}
```

**使用**：
```java
@Service
public class UserService {
    
    @Async("taskExecutor")
    public void asyncUpdateLoginInfo(Long userId, String ip) {
        // 异步更新登录信息
    }
}
```
