## 二、业务主链（用户视角）

```text
认证登录
  → 用户资料与关系
  → 团队协作
  → 聊天与消息体验
```

## 三、平台能力（治理与智能）

```text
平台治理：举报 → 处罚 → 申诉 → 撤销
智能能力：搜索 → 推荐 → 点击反馈 → 刷新
安全能力：设备管理 → 风控扩展（已并入平台治理）
```

---

## 四、关键横向能力

- 统一管理员鉴权：`AdminAuthService` + `t_admin_user`
- 统一缓存策略：搜索热词、推荐结果
- 统一通知链路：审核、处罚、申诉结果
- 统一安全边界：黑名单、隐私、处罚状态
