## 7. 模块测试策略
### 7.1 登录认证
链路：验证码 -> 登录 -> 获取当前用户；注册 -> 邮箱验证码 -> 注册；忘记密码 -> 新密码登录。  
异常：错误密码、错误验证码、邮箱已注册、验证码过期、未登录访问 `/api/auth/me`。

### 7.2 用户资料与隐私
链路：修改头像/昵称/简介 -> 重新获取资料 -> 页面同步；修改隐私设置 -> 其他账号访问验证。  
异常：昵称超长、tags 非法、访问不存在用户、未登录修改资料。

### 7.3 好友关系
链路：发起申请 -> 查看申请 -> 同意/拒绝 -> 列表更新 -> 删除好友。  
异常：不能加自己、重复申请、已是好友重复申请、非接收方审批。

### 7.4 黑名单
链路：拉黑 -> 验证好友/私聊受影响 -> 解除拉黑 -> 能力恢复。  
异常：重复拉黑、解除未拉黑对象、拉黑不存在用户。

### 7.5 团队
链路：创建团队 -> 申请/密码加入 -> 审批 -> 成员列表 -> 修改角色 -> 转让队长 -> 全员禁言/成员禁言 -> 退出/解散。  
异常：密码错误、满员加入、普通成员审批失败、普通成员禁言失败、已解散团队继续操作。

### 7.6 聊天
链路：

+ 私聊：发送 -> 历史 -> 编辑 -> 撤回 -> 收藏 -> 置顶 -> 搜索 -> 已读
+ 群聊：发送 -> 历史 -> 公告 -> 兼容禁言接口
+ 举报：举报 -> 查询状态 -> 管理员查看上下文 -> 管理员处理 -> 发起申诉  
异常：空消息、非发送者编辑/撤回、非成员发群消息、举报不存在消息、非管理员访问管理接口、已禁言成员仍可发言。

### 7.7 OSS 上传
链路：获取预签名 -> 实际上传 -> URL 可访问 -> 头像/消息中可用。  
异常：文件为空、类型非法、文件过大、token 过期。

### 7.8 搜索与推荐
链路：搜索 -> 搜索历史 -> 热搜 -> 联想 -> 推荐列表 -> 点击推荐上报。  
异常：空关键词、超长关键词、非法 `type/searchType`、推荐点击非法参数。  
说明：推荐点击接口测试应按当前真实实现字段执行。

### 7.9 系统通知
链路：业务动作触发通知 -> 未读数变化 -> 列表查询 -> 标已读 -> 删除。

### 7.10 设备管理
链路：绑定设备 -> 查询列表 -> 信任设备 -> 下线 -> 删除设备。  
异常：重复绑定、删除不存在设备、操作非本人设备。

### 7.11 在线状态
链路：设置状态 -> 心跳 -> 查询状态 -> 主动下线。

### 7.12 举报/申诉/反馈/处罚
要求覆盖完整业务闭环：提交 -> 查询/列表 -> 管理员处理 -> 用户查看结果/恢复状态。  
异常：越权处理、重复处理、资源不存在。
