# 测试账号与演示数据说明

## 一、建议准备的账号类型

### 1. 普通用户
用于演示：
- 注册登录
- 编辑资料
- 搜索用户/团队
- 私聊/群聊
- 收藏/置顶/举报/反馈

### 2. 队长账号
用于演示：
- 创建团队
- 修改团队
- 审批入队
- 设置管理员
- 移除成员
- 设置群公告

### 3. 管理员账号
用于演示：
- 平台处罚
- 审核用户举报 / 团队举报 / 消息举报
- 处理申诉
- 处理反馈

## 二、建议准备的数据

- 1 个公开团队
- 1 个私有团队
- 2~3 个普通成员
- 若干聊天消息
- 若干系统通知
- 若干举报/反馈记录

## 三、前端演示建议顺序

1. 登录
2. 搜索用户和团队
3. 创建团队 / 申请入队
4. 私聊 / 群聊
5. 编辑 / 撤回 / 收藏 / 置顶
6. 公告 / 举报 / 申诉 / 反馈
7. 推荐与热词
