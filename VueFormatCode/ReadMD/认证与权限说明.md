# 认证与权限说明

## 一、登录态

- 登录成功后前端保存 token
- 所有需鉴权接口统一携带 token
- token 失效后前端统一跳转登录页

## 二、匿名接口

- `GET /api/auth/captcha`
- `POST /api/auth/code`
- `POST /api/auth/register`
- `POST /api/auth/login`

## 三、普通登录用户可访问

- 用户资料管理
- 好友与黑名单
- 团队查询、申请、聊天、搜索、推荐
- 举报、申诉、反馈、设备管理

## 四、团队角色权限

### 1. 队长
- 更新团队资料
- 解散团队
- 审核入队申请
- 设置管理员 / 取消管理员
- 转移队长
- 移除成员
- 设置全员禁言
- 设置群公告

### 2. 管理员
- 审核入队申请
- 移除普通成员
- 禁言普通成员
- 设置群公告

### 3. 普通成员
- 查看团队
- 发送群消息
- 查看群公告
- 正常使用搜索与推荐

> 重要：已取消嘉宾角色，前端不要再实现嘉宾权限分支。

## 五、管理员后台权限

平台管理员可访问：
- 平台处罚
- 用户举报审核
- 团队举报审核
- 消息举报审核
- 平台申诉审核
- 用户反馈处理

## 六、常见权限失败前端处理

- 未登录：跳转登录
- 无权限：Toast 提示“无权限操作”
- 被处罚/禁言：直接展示后端返回的 message
